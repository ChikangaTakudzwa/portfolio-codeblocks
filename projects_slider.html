  • E Commerce Mobile Phone Store

    Introducing a Mobile Shop E-Commerce project, a modern online shopping experience built with Django, Tailwind, and HTMX technologies. This platform enables users to effortlessly explore and purchase mobile phones and accessories from the comfort of their homes. The website features a responsive design, ensuring seamless navigation across various devices.

    Key features include secure user authentication, a robust product catalog with search and filtering options, detailed product pages with reviews, a streamlined shopping process with Stripe integration, and an admin dashboard for efficient management of products and orders.

    The backend is powered by the Django web framework, providing stability and scalability. Tailwind CSS enhances the frontend with a sleek and modern layout, while HTMX adds dynamic and interactive components. Data is stored efficiently using PostgreSQL on bit.io, and secure payments are facilitated through the Stripe API.

    Project Type: E-commerce Website
    Tech Stack: Django, TailwindCSS, PostgreSQL, Stripe, HTML & HTMX

  • ResumeForge

    Introducing Resumeforge, an innovative web-based resume generator crafted with a powerful tech stack that includes Bubble.io, Express, Node.js, and harnesses the capabilities of the OpenAI API. This application redefines the resume creation process by seamlessly transforming user-submitted data into compelling and professional resume content

    At the heart of Resumeforge is a user-friendly form where individuals input their relevant information. Leveraging the prowess of natural language processing algorithms, the application goes beyond conventional resume builders. It intelligently interprets the user's input to generate a customized resume that not only showcases skills, experience, and accomplishments but also crafts them into a narrative that resonates professionally.

    Project Type: AI Resume Content Generator
    Tech Stack: Bubble.io, OpenAI, NodeJs, ExpressJs & Render.io
